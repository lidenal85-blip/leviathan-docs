    1. Найти задачу в чек-листе
    2. Изменить статус:
    3. [ ] → [~] (в процессе)
    4. [~] → [x] (выполнено)

    5. [x] → [!] (ошибка/откат)

    6. Заполнить метаданные: 

      - [x] **1.1** 🔴 Создать структуру LAP

  - **Результат:** Создано 5 директорий

    7. Обновить шапку документа:
