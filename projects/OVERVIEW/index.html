    🏗️ LEVIATHAN Platform Architecture

    Version: 5.0 | Last Updated: 2026-07-03

    Executive Summary

    LEVIATHAN — это AI-ориентированная платформа, объединяющая 58 проектов на одном сервере (78.17.24.96). Система состоит из центрального AI-движка (leviathan_engine), роя AI-агентов (Claude, Gemini, Hermes), бизнес-сервисов (Arbitr, SavdaBot) и веб-платформ (Kinovibe, Voicestudio).

    Core Components

    1. AI Engine Layer

    • leviathan_engine — Central orchestrator with agent_service, MCP server, core library
    • den4ik-claude — Claude AI session manager with pool of accounts
    • hermes_agent — Hermes autonomous agent with MCP client capabilities
    • triton_agent — Specialized AI agent for specific tasks

    2. Business Layer

    • arbitr_cockpit — Business intelligence platform with pipeline system
    • savdabot — Telegram-based service bot
    • diet_platform — Diet planning service
    • pelleto-ai — Android/AI assistant for Termux

    3. Web Layer

    • kinovibe — Movie discovery platform
    • voicestudio — Voice processing studio
    • vocalplatform — Voice platform API
    • f3platform — Web platform

    4. Infrastructure

    • secrets_vault — Centralized secrets management
    • leviathan-docs — This documentation site
    • 25+ systemd services managed via systemctl

    Architecture Principles

    1. AI-native — Claude and Gemini as core computational resources
    2. Microservice-like — Each service is an independent systemd unit
    3. Unified MCP — Model Context Protocol for tool interoperability
    4. Config-driven — Environment files and JSON configs for all services
