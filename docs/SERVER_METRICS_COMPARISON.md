### Findings

1. **freebuff — главный потребитель**: 19.7% RAM + 44.6% CPU. Требует оптимизации.
2. **API ключи невалидны**: Groq возвращает 401 Invalid API Key. Все ключи в cooldown.
3. **CircuitBreaker разомкнут**: Для gemini и groq после 3+ неудачных попыток.
4. **Load 8.37 на 2 ядра**: Очередь процессов более 4х кратно превышает мощность CPU.

### Recommendations

- Перезагрузить/обновить API-ключи в secrets-vault
- Оптимизировать freebuff (или перезапустить)
- Рассмотреть добавление ядер или миграцию процессов
