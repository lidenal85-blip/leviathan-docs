## 🔄 ПРОТОКОЛ ЛОГИРОВАНИЯ

### Формат записи:

```markdown
### HH:MM — Краткое описание
- **Действие:** Что было сделано
- **Файлы:** Какие файлы изменены/созданы
- **Команда:** Какие команды выполнены
- **Результат:** ✅ Успех / ❌ Ошибка (описание)
- **Проблемы:** Если были проблемы — как решены
```

### Правила:
1. **Каждое действие** записывается в лог
2. **Таймстамп** в формате HH:MM (UTC)
3. **Дата** в заголовке `## 📅 YYYY-MM-DD`
4. **Файлы** перечисляются списком
5. **Команды** указываются ключевые
6. **Результат** — чёткий статус (✅/❌)
7. **Проблемы** — если были, описать решение

### Как добавить запись:

```bash
lev 'cd /opt/leviathan-docs && ./add_log.sh "HH:MM" "Описание" "Файлы" "Команда" "Результат"'
```
