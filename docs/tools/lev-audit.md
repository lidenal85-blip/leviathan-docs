# lev-audit — Project Audit

Быстрый аудит проектов LEVIATHAN с Tier-классификацией.

## Usage

```bash
lev-cli audit [--json] [--project <name>]
```

## Tiers

| Tier | Description | Examples |
|------|-------------|---------|
| 1 🔴 | Critical | leviathan_engine, den4ik, arbitr |
| 2 🟡 | Important | savda, diet, pelleto |
| 3 🟢 | Standard | vocal, kwork, fashion |
| 4 ⚪ | Minor | Other projects |
