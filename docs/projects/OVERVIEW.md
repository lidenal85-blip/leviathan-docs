## Architecture Principles

1. **AI-native** — Claude and Gemini as core computational resources
2. **Microservice-like** — Each service is an independent systemd unit
3. **Unified MCP** — Model Context Protocol for tool interoperability
4. **Config-driven** — Environment files and JSON configs for all services
